# Zova

Zova 是一款支持 IOC 容器的 Vue3 框架。有了 IOC 容器的加持，定义响应式状态不再需要`ref/reactive`，也不再需要`ref.value`

```bash
$ npm run init
$ cd ./zova-dev
$ npm run dev
```
