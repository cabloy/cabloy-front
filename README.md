# Zova

Zova is a vue3 framework with ioc container. With the support of ioc container, defining reactive states no longer needs `ref/reactive`, nor `ref.value`

## How to do

```bash
$ pnpm install
$ cd ./zova-dev
$ pnpm install
$ npm run dev
```
